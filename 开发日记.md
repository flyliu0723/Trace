# SpendWhere 开发日记

## v0.1.1 · 2026-07-03

### 计步运动检测（国内可用）

- 新增 `StepCounterWatcher`：硬件 `TYPE_STEP_COUNTER` 每分钟采样，不依赖 Google Play Services
- 步数增量 ≥20/分钟 → `activity_change: WALKING`；连续低活动 → `STILL`
- 与 Google Activity Recognition 并行运行，事件带 `metadata.detector`（`step_counter` / `google`）
- 监控状态新增：计步传感器可用性、运行状态、Google 服务可用性
- 行走使用手机分析扩展为识别 `WALKING` / `RUNNING` / `ON_FOOT`
- Google Activity Recognition 补充监听 `ON_FOOT`

## v0.1.0 · 2026-07-03

**现代精细主义视觉重组** — 让数据呈现更有呼吸感，降低空态挫败感。

### 导航与设置

- 底部 Tab 精简为 **轨迹 / 洞察 / 时间线**，设置移入各页齿轮入口
- 设置页 grouped list 重构：MonitorBanner（可关闭 + 持久化）、权限折叠卡片、SegmentControl 滑动指示器
- 采集/权限异常时齿轮按钮显示状态点提示

### 轨迹（首页）

- 冷灰底 `#F4F6F8` + 白卡片 + 弥散微投影
- 解锁热力图：更小圆环、莫兰迪紫胶囊周柱、星宿留白风格

- MonitorBanner 文案与 spring 按压反馈优化

### 洞察

- **AI 洞察未配置态**：保留周期切换栏，空态区改用清透渐变微光背景；钥匙图标缩小，温和产品导向文案；未配置时隐藏分享/重写

### 时间线
